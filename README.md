# Desafio Prático - CortechX 2026.2

Resolução do desafio técnico para o processo seletivo da **CortechX**. O notebook engloba desafios práticos resolvidos nas frentes de Engenharia de Dados, Análise de Dados, Visão Computacional e Automação (Agentes Inteligentes).

## 📁 Estrutura do Repositório

- `data/`: Datasets utilizados nas questões.
- `models/`: Modelos de Visão Computacional.
- `utils/`: Scripts auxiliares e lógica modularizada (`gesture_utils.py` e `pokemon_utils.py`).
- `*.ipynb`: Notebook principal com as respostas e análises.

## ⚙️ Execução

Para rodar o notebook localmente ou no Google Colab, certifique-se de configurar a variável de ambiente ou secret `GEMINI_API_KEY` para o funcionamento do pipeline de automação (Questão 7).
